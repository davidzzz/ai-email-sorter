# AI Email Sorter

Run locally (requires Docker):

1. Copy `.env.example` to `.env` and fill in keys.
2. Run `docker-compose up --build`
3. In the frontend, run `npm run preview`
4. Open `http://localhost:3000` (frontend). Backend on `http://localhost:4000`.
5. For testing purposes, run `npm test`. Backend and frontend use the same testing command.
